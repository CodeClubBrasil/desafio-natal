Desafio de Natal

E aí, já pensou ter um jogo seu como lição do Code Club Brasil?


Sobre o desafio

Olá, voluntário!

De 17 de novembro a 12 de dezembro de 2014 você (sozinho ou junto com seu clube) poderá submeter uma proposta de jogo desenvolvido no Scratch. O jogo mais curtido e compartilhado até o dia 31 de dezembro servirá como base para a primeira lição criada por um Voluntário/ Clube do Code Club Brasil.

Os Critérios

Jogo deve ser feito no Scratch online
O jogo deve utilizar o tema Natal com jeitinho Brasileiro.
O jogo deve utilizar elementos do KIT disponibilizado no ato da inscrição
Os participantes podem participar com mais de um jogo.

Então, é a sua chance de colocar toda a criatividade para fora, caprichar na mecânica, no visual do jogo e dar mais uma super contribuição para a construção de um futuro fantástico para as crianças participantes do Code Club Brasil.
